# 에브리타임+ (내가 만든 학점 계산기)
## 무엇을 하는 프로그램인가요?
#### 대학생들이 자신의 수강 과목과 성적을 체계적으로 기록하고 관리할 수 있도록 지원하는 명령줄 인터페이스(CLI) 기반의 학점 관리 및 시뮬레이션 프로그램입니다. 단순히 현재 평점을 계산하는 것을 넘어 학칙에 맞는 영역별 이수 현황을 대시보드로 브리핑하고, 사용자가 설정한 미래의 목표 졸업 평점을 달성하기 위해 남은 학기 동안 획득해야 하는 점수를 수학적으로 역산해 주는 스마트 가이드라인 기능을 제공합니다.

## 실행 방법
#### Python 3.10 이상 설치 (프로그램 안정성을 위해 최신 버전의 인터프리터 권장)

#### 터미널 또는 명령 프롬프트(CMD)를 열고 프로그램 파일이 있는 폴더로 이동

#### 다음 명령어를 입력하여 실행:

#### Bash
#### python main.py
## 사용된 Python 개념
#### 리스트(List) & 딕셔너리(Dictionary): 과목 데이터를 구조화된 딕셔너리로 다루고, 전체 수강 내역을 리스트 객체에 누적하여 관리

#### #### 람다식(Lambda) 및 정렬(Sorted): 학기 및 이수 구분 우선순위에 따른 다중 조건 정렬 기능 구현

#### 예외 처리 (Try-Except): 사용자 데이터 입력 시 발생하는 타입 에러(ValueError) 및 0으로 나누기(ZeroDivisionError) 방어

#### 파일 입출력 (JSON 파일 연동): json 내장 모듈을 활용하여 프로그램 종료 후에도 데이터가 유지되는 영속성 레이어(grades.json) 구현

## AI 도구 사용 내역
#### 의사코드(Pseudocode) 설계 구체화: 추상적인 기능 흐름을 파이썬 함수 형태의 의사코드로 명확히 구조화하는 데 프롬프팅을 활용함

#### 디버깅 및 트러블슈팅: 데이터 정렬 조회 시 원본 리스트와 출력 리스트 간의 인덱스 불일치로 발생하던 심각한 과목 삭제 오류를 논리적으로 분석하고, remove() 메서드를 통한 객체 직접 타깃팅 방식으로 코드를 정교하게 개선함

#### 문서화: 트러블슈팅 및 시행착오 과정을 줄글 형태의 제출용 보고서 양식과 기술용 README 서식으로 정제하는 데 협업함

## 한계와 다음 단계
#### 아직 안 되는 것: 현재는 명령줄(CLI) 환경에서 텍스트로만 구동되기 때문에 사용자가 직관적으로 다루기에는 시각적 한계가 있습니다. 또한 F학점의 경우 평점 계산에는 포함되지만 이수 완료 학점에서는 제외되어야 하는 세부 학칙이 대시보드 누적 로직에 완벽히 분리되지 않아 정교함이 다소 부족합니다.

#### 더 하고 싶은 것: Tkinter나 PyQt 라이브러리를 도입하여 에브리타임 서비스와 유사한 깔끔한 그래픽 사용자 인터페이스(GUI)를 입히고 싶습니다. 더불어 누적 성적 변화 추이를 꺾은선 그래프로 시각화하는 기능과 F학점 과목에 대한 재수강 여부를 선택하여 이전 성적을 대체하는 학칙 예외 로직을 추가하고 싶습니다.
